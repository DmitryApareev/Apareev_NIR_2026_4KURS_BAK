# Apareev_NIR_2026_4KURS_BAK

## Программный комплекс для оценки кредитного риска на основе алгоритмов машинного обучения.

Проект реализует прототип системы кредитного скоринга, позволяющий загружать данные, выполнять предобработку признаков, обучать несколько моделей классификации и сравнивать их качество.

## Реализованный функционал
- загрузка данных из CSV-файлов
- предобработка числовых и категориальных признаков
- обучение моделей:
  - Logistic Regression
  - Random Forest
  - Gradient Boosting
- расчет метрик качества:
  - ROC-AUC
  - F1-score
- построение ROC-кривых
- сохранение результатов в файл

## Структура проекта
- `data/` — входные датасеты
- `src/` — исходный код модулей
- `outputs/` — результаты работы программы
- `tests/` — тесты
- `main.py` — основной файл запуска

## Установка
1. Клонировать репозиторий:
```bash
git clone https://github.com/DmitryApareev/Apareev_NIR_2026_4KURS_BAK
cd Apareev_NIR_2026_4KURS_BAK
```

2. Установить зависимости:
```bash
pip install -r requirements.txt
```

3. Запуск
```bash
python main.py
```
